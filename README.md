

#### 代码生成转换工具

- 期望（目标）

      1：将java代码转化为其他语言的代码

- 实现思路

      1：通过扫描java类生成的统一java类描述元数据


- 新的期望

     1：可以通过java代码或 open Api 规范进行生成 (java =>any => any => any)

- 新的思路

     1：增加通过解析open api的json格式文档，构建统一的描述元数据
     2：默认提供统一解析，通过提供不同语言的解析器进行自定义的处理
     3：在模板层面进行抽象，可以实现根据不同的数据分发不同的模板（灵活的模板加载策略）
